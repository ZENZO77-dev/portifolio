# M. M. Zendera — GIS Analyst & Surveyor Portfolio

This repository contains a static portfolio site for M. M. Zendera showcasing experience in engineering surveying, remote sensing, and GIS.

Contents
- `index.html` — main site
- `styles.css` — styling
- `script.js` — small interactions (image preview, filters, smooth scroll)
- `profile-placeholder.jpg` — placeholder profile image (replace with your photo)

How to run locally
1. Clone the repo
2. Open `index.html` in your browser, or run a simple server:
   - `python -m http.server` and open http://localhost:8000

Deploy
- Push to GitHub and enable GitHub Pages (deploy from `main` branch or `docs/` folder)
- Or host on Netlify/Vercel/any static host

Notes
- The Change photo button lets you upload a profile image locally (it will not persist to the repo unless you add the file to the repository).
- Update contact details, project descriptions, and images as needed.
